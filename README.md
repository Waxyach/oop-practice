### 🛠 Технологічний стек

Для забезпечення якості коду та автоматизації використовуються наступні інструменти:

| Інструмент |                  Призначення                  |
|:-----|:---------------------------------------------:|
| **Lombok** |         Мінімізація бойлерплейт-коду          |
| **Axion** | Автоматичне керування версіями через Git теги |
| **Spotless** | Форматування коду згідно з **Palantir style** |
| **JUnit 5** |       Модульне тестування бізнес-логіки       |

-----

### 📂 Структура проєкту

Проєкт організований як мультимодульна система Gradle:

```text
.
├── build-logic          # Спільні конвенції збірки (my-conventions)
├── task1                # Завдання 1: Розрахунок струмів
└── build.gradle.kts     # Головний файл конфігурації
```

### 📝 Виконані завдання

#### **Завдання 1: Розрахунок струмів у паралельних опорах**

Розробка системи для обчислення струму, що проходить крізь чотири паралельно підключені опори.

* **Реалізовано:** серіалізація об'єктів, використання `transient` полів для оптимізації.
* **Тестування:** повне покриття логіки за допомогою JUnit.
